About This Blog

Welcome to the Simple Blog! This blog was created as a demonstration of a Django blog with pagination, search, and comment functionality.

Features

  • 📝 Create, edit, and delete blog posts - Admin only
  • 🔍 Search posts - By title or content
  • 🏷️ Categories - Organize posts by topic
  • 📄 Pagination - 5 posts per page for better navigation
  • 💬 Comment system - Readers can leave comments
  • 📊 Post view counter - Track popularity
  • 📱 Responsive design - Works on all devices

Technologies Used

  • 🐍 Django 5.2 - Python Web Framework
  • 🎨 Bootstrap 5 - Frontend Framework
  • 🗄️ SQLite - Database
  • 🎯 Font Awesome 6 - Icons

How to Use

  1. Login as admin to create new posts
  2. Browse posts by category or search
  3. Read posts and leave comments
  4. Share posts on social media
Contact: Have questions or suggestions? Feel free to leave a comment on any post!
