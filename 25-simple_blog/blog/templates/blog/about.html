About Simple Blog

Welcome to Simple Blog! This is a modern blog built with Django, featuring all the essential functionality you'd expect from a professional blog platform.

Features

  • Pagination - Browse posts page by page
  • Search - Find posts by title or content
  • Categories - Organize content by topics
  • Tags - Add relevant tags to posts
  • Comments - Engage with readers
  • Likes - Show appreciation for posts
  • Markdown Support - Write formatted content
  • Newsletter - Subscribe for updates

Technologies Used

  • Django 5.2 - Python Web Framework
  • Bootstrap 5 - Frontend Framework
  • SQLite - Database (can be switched to PostgreSQL)
  • Font Awesome 6 - Icons
  • Markdown - Content formatting

Getting Started

  1. Login as admin at /admin/
  2. Create categories and tags
  3. Start writing blog posts
  4. Manage comments and engage with readers
Questions? Feel free to leave a comment on any post!
